
In this folder you will find:

  GridWorld.h : the skeleton into which you will place
    your solution

  driver.cpp : a simple interactive driver program that
    lets you play around with your GridWorld.h implementation

    to compile:  g++ -std=c++11 driver.cpp -o driver

  TESTS : folder with a test suite to stress test (and score)
    your solution.  See the README.txt file in the folder
